Clausen Functions

The Clausen function is defined by the following integral,
Cl_2(x) = - \int_0^x dt \log(2 \sin(t/2))
It is related to the dilogarithm by Cl_2(\theta) = \Im Li_2(\exp(i \theta)).
GSL::Special::Clausen::clausen
GSL::Special::Clausen::clausen_e
Clausen::clausen(d1) -> d2
These routines compute the Clausen integral Cl_2(x). 
require "GSL"; include GSL
include GSL::Special

x = 1.7
r = Clausen::clausen(x)
p r
r = Clausen::clausen_e(x)
p r.val
p r.err
