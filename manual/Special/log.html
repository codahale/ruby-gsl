Logarithm and Related Functions

Information on the properties of the Logarithm function can be found in Abramowitz & Stegun, Chapter 4.

Module Log

GSL::Special::Log::log
GSL::Special::Log::log_e
Log::log(x) -> d1
x is a double. These routines compute the logarithm of x, \log(x), for x > 0.

GSL::Special::Log::log_abs
GSL::Special::Log::log_abs_e
Log::log_abs(x) -> d1
These routines compute the logarithm of the magnitude of x, \log(|x|), for x \ne 0.

GSL::Special::Log::complex_log
MISSING; see Complex_log
This routine computes the complex logarithm of z = z_r + i z_i. The results are returned as lnr, theta such that \exp(lnr + i \theta) = z_r + i z_i, where \theta lies in the range [-\pi,\pi].

GSL::Special::Log::log_1px_mx
GSL::Special::Log::log_1px_mx_e
Log::log_1px_mx(x) -> d1
These routines compute \log(1 + x) for x > -1 using an algorithm that is accurate for small x. (see also Math::log1p).

GSL::Special::Log::log_1px_mx
GSL::Special::Log::log_1px_mx_e
Log::log_1px_mx(x) -> d1
These routines compute \log(1 + x) - x for x > -1 using an algorithm that is accurate for small x. 

require 'assert'
require "GSL"
include GSL
include GSL::Special
include Math

puts "log"
x = 7.7
r = Log::log(x)
assert r == log(x)

puts "log_abs"
r = Log::log_abs(-x)
assert r == log(x)


puts "log_1px" 
x = 1 + 1.0e-10
r = Log::log_1px(x)
assert r == Math::log1p(x) && r == log(1 + x)

r = Log::log_1px_e(x)
p r.val; p r.err
  
puts "log_1px_mx"
x = 1 - 1.0e-10
r = Log::log_1px_mx(x)
assert r == log(1 + x) - x
