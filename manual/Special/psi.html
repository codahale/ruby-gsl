Psi (Digamma) Function

The polygamma functions of order m defined by \psi^{(m)}(x) = (d/dx)^m \psi(x) = (d/dx)^{m+1} \log(\Gamma(x)), where \psi(x) = \Gamma'(x)/\Gamma(x) is known as the digamma function. See Abramowitz and Stegun, chapter 6.

Digamma Function

GSL::Special::Psi::psi_int
GSL::Special::Psi::psi_int_e
Psi::psi_int(n) -> d1
n is an integer. These routines compute the digamma function \psi(n) for positive integer n. The digamma function is also called the Psi function.

GSL::Special::Psi::psi
GSL::Special::Psi::psi_e
Psi::psi(x) -> d1
x is a double. These routines compute the digamma function \psi(x) for general x, x \ne 0.

GSL::Special::Psi::psi_1piy
GSL::Special::Psi::psi_1piy_e
Psi::psi_1piy(x) -> d1
These routines compute the real part of the digamma function on the line 1+i y, Re[\psi(1 + i y)].

Trigamma Function

GSL::Special::Psi::psi_1_int
GSL::Special::Psi::psi_1_int_e
Psi::psi_1_int(n) -> d1
n is an integer. These routines compute the Trigamma function \psi'(n) for positive integer n.

Polygamma Function

GSL::Special::Log::psi_n
GSL::Special::Log::psi_n_e
Log::psi_n(n, x) -> d1
These routines compute the polygamma function \psi^{(m)}(x) for m >= 0, x > 0.

p r

r = Psi::psi(x)
assert r == Psi::psi_int(x)
x = 69.2
r = Psi::psi(x)
p r
r = Psi::psi_e(x)
p r.val; p r.err

puts "psi_1piy"
y = 1
r = Psi::psi_1piy(y)
p r

puts "psi_1_int"
x = 1
r = Psi::psi_1_int(x)
# wrong sign
assert r ==  -M_PI*M_PI/6.0

x = 2
r = Psi::psi_1_int(x)
assert r == M_PI*M_PI/6 - 1

x = 3
r = Psi::psi_1_int(x)
assert fcmp(r, M_PI*M_PI/6 - 5.0/4, eps) == 0

x = 4
r = Psi::psi_1_int(x)
assert fcmp(r, M_PI*M_PI/6 - 49.0/36, eps) == 0

x = 110
r = Psi::psi_1_int(x)
p r

x = 1
r = Psi::psi_n(1, x)
assert r == M_PI*M_PI/6

(2..100).each do |i| 
  assert(fcmp(Psi::psi_n(1,i), Psi::psi_1_int(i), eps) == 0) 
end
