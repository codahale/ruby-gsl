Trigonometric Functions

The library includes its own trigonometric functions in order to provide consistency across platforms and reliable error estimates.

GSL::Special::Trig::hypot
GSL::Special::Trig::hypot_e
Trig::hypot(x, y) -> d1
x and y are doubles. These routines compute the hypotenuse function \sqrt{x^2 + y^2} avoiding overflow and underflow.

GSL::Special::Trig::sinc
GSL::Special::Trig::sinc_e
Trig::sinc(x) -> d1
These routines compute \sinc(x) = \sin(\pi x) / (\pi x) for any value of x.

Conversion Functions

GSL::Special::Trig::polar2rect
Trig::polar2rect(r, theta) -> anArray
This function converts the polar coordinates (r,theta) to rectilinear coordinates (x,y), x = r\cos(\theta), y = r\sin(\theta). It returns a two-dimensional array of Result objects.

GSL::Special::Trig::rect2polar
Trig::rect2polar(x, y) -> anArray
This function converts the rectilinear coordinates (x,y) to polar coordinates (r,theta), such that x = r\cos(\theta), y = r\sin(\theta). It returns a two-dimensional array of Result objects. The argument theta lies in the range [-\pi, \pi].

Restriction Functions

GSL::Special::Trig::angle_restrict_symm
Trig::angle_restrict_symm(theta) ->
These routines force the angle theta to lie in the range (-\pi,\pi].

GSL::Special::Trig::angle_restrict_pos
Trig::angle_restrict_pos(theta) ->
These routines force the angle theta to lie in the range [0, 2\pi)

Trigonometric Functions With Error Estimates

GSL::Special::Trig::sin_err_e
Trig::sin_err_e(x, dx) -> res
These routines compute the sine of an angle x with an associated absolute error dx, \sin(x \pm dx). It returns an object of typr Result.

GSL::Special::Trig::cos_err_e
Trig::cos_err_e(x, dx) -> res
These routines compute the cosine of an angle x with an associated absolute error dx, \cos(x \pm dx). 

require 'assert'
require "GSL"; include GSL
include GSL::Special; include Math
require 'gsl_util'

eps = 1.0e-15

puts "sin"
x = 0.987
assert Trig::sin(x) == sin(x)

puts "cos"
assert fcmp(Trig::cos(x), cos(x), eps) == 0

puts "hypot"
y = 0.3
assert Trig::hypot(x, y) == hypot(x, y)

puts "sinc"
assert fcmp(Trig::sinc(x), sin(M_PI*x)/(M_PI*x), eps) == 0

puts "complex_sin_e"
y = 0.3
r = Trig::complex_sin_e(x, y)
c1 = Complex.new2(x, y)
assert r[0].val == Complex.sin(c1).real &&  r[1].val == Complex.sin(c1).imag

puts "complex_cos_e"
r = Trig::complex_cos_e(x, y)
assert r[0].val == Complex.cos(c1).real &&  r[1].val == Complex.cos(c1).imag

puts "complex_logsin_e"
y = 0.3
r = Trig::complex_logsin_e(x, y)
c1 = Complex.new2(x, y)
assert r[0].val =~ Complex.logsin_e(c1)[0].val
assert r[0].val ==  Complex.logsin_e(c1)[0].val &&  r[1].val ==  Complex.logsin_e(c1)[1].val

puts "lnsinh"
r = Trig::lnsinh(x)
assert r == log(Complex::sinh(Complex.new2(x,0)).real)

puts "lncosh"
r = Trig::lncosh(x)
assert r == log(Complex::cosh(Complex.new2(x,0)).real)
p r
r = Trig::lncosh_e(x)
p r.val; r.err

puts "polar2rect"
radius = 1.0; theta = M_PI/6
ary = Trig::polar2rect(radius, theta)
assert ary[0].val == radius * cos(theta)
assert ary[1].val == radius * sin(theta)

puts "rect2polar"
ary = Trig::rect2polar(radius * cos(theta), radius * sin(theta))
assert ary[0].val == radius
assert ary[1].val == theta

puts "angle_restrict_pos"
r = Trig::angle_restrict_pos(theta + 2*M_PI)
assert fcmp(r,theta, eps) == 0

puts "sin_err_e"
r = Trig::sin_err_e(x,1.0e-18)
r1 = r.val; p r.err
r = Trig::sin_err_e(x,1.0e-14)
r2 = r.val; p r.err
assert r1 == r2 && r1 == sin(x)

puts "cos_err_e"
r = Trig::cos_err_e(x,1.0e-18)
r1 = r.val; p r.err
assert r1 =~ cos(x)
