Error functions

Error functions

The error function is described in Abramowitz & Stegun, Chapter 7.

Module Error

Error Function

GSL::Special::Error::erf
GSL::Special::Error::erf_e
Error::erf(d1) -> d2
These routines compute the error function erf(x) = (2/\sqrt(\pi)) \int_0^x dt \exp(-t^2).

Complementary Error Function

GSL::Special::Error::erfc
GSL::Special::Error::erfc_e
Error::erfc(d1) -> d2
These routines compute the complementary error function erfc(x) = 1 - erf(x) = (2/\sqrt(\pi)) \int_x^\infty \exp(-t^2).

Log Complementary Error Function

GSL::Special::Error::log_erf
GSL::Special::Error::log_erf_e
Error::log_erf(d1) -> d2
These routines compute the logarithm of the complementary error function \log(\erfc(x)).

Probability functions

The probability functions for the Normal or Gaussian distribution are described in Abramowitz & Stegun, Section 26.2. GSL::Special::Error::erf_Z
GSL::Special::Error::erf_Z_e
GSL::Special::Error::erf_Z(d1) -> d2
These routines compute the Gaussian probability function Z(x) = (1/\sqrt(2\pi)) \exp(-x^2/2).

GSL::Special::Error::erf_Q
GSL::Special::Error::erf_Q_e
GSL::Special::Error::erf_Q(d1) -> d2
These routines compute the upper tail of the Gaussian probability function
Q(x) = (1/\sqrt(2\pi)) \int_x^\infty dt \exp(-t^2/2). 

require 'assert'
require "GSL"
include GSL
include GSL::Special
include Math


eps = 1.0e-15

puts "Error function"
x = 0.5
r = Error::erf(x)
p r
r = Error::erf_e(x)
p r.val; p r.err


puts "Complementary Error Function"
x = 2.0
r = Error::erfc(x)
p r * x * exp(x*x)
assert float_equal(r, 1 - Error::erf(x), eps/10)

puts "Log Complementary Error Function"
r = Error::log_erf(x)
assert r == log(Error::erfc(x))

puts "Probability functions"
r = Error::erf_Z(x)
assert float_equal(r, 1/sqrt(2*M_PI) * exp(-x*x/2), eps)

r = Error::erf_Q(x)
p 1 - r
