Elementary Operations

The following functions allow for the propagation of errors when combining quantities by multiplication.

Module Elementary

GSL::Special.Elementary#multiply
 Elementary#multiply(d1, d2) -> d3
 GSL::Special.Elementary#multiply_e
 Elementary#multiply_e(d1, d2) -> res
 x and y are doubles. This function multiplies x and y storing the product and its associated error in res, which belongs to the class Result.

GSL::Special.Elementary#multiply_err_e
 Elementary#multiply_err_e(res1, res2) -> res3
 This function multiplies x and y with associated absolute errors dx and dy. res1 is a Result object for a double x with error dx; res2 is a Result object for a double y with error dy. The product xy +/- xy \sqrt((dx/x)^2 +(dy/y)^2) is stored in res3. 

require 'assert'
require "GSL"
include GSL
include GSL::Special

x = 0.5; y = 0.9
p Elementary::multiply(x,y)
res = Elementary::multiply_e(x,y)
p res.val; p res.err

puts "  multiply_err_e"
r1 = Debye::D1_e(x)
r2 = Dilog::dilog_e(x)
printf "r1      : %20.18f %g\n", r1.val, r1.err
printf "r2      : %20.18f %g\n", r2.val, r2.err
assert r1.class == GSL::Result && r1.class == GSL::Result
r = Elementary::multiply_err_e(r1.val, r1.err, r2.val, r2.err)
printf "r1 * r2 : %20.18f %g\n", r.val, r.err
