Elliptic Functions (Jacobi)

The Jacobian Elliptic functions are defined in Abramowitz & Stegun, Chapter 16.

Module EllJac

GSL::Special::EllJac::e
EllJac::e(d1, d2) -> anArray
This function computes the Jacobian elliptic functions sn(u|m), cn(u|m), dn(u|m) by descending Landen transformations. It returns the array [sn(u|m), cn(u|m), dn(u|m)] . 
require "GSL"
include GSL
include GSL::Special

sn, cn, dn = EllJac::e(10.2, 0.2)
print "  e: ", sn, " ", cn, " ", dn, "\n"

sn, cn, dn = EllJac::e(0.19, 1.0/(19*19))
print "  e: ", sn, " ", cn, " ", dn, "\n"

sn, cn, dn = EllJac::e(0.20, 0.19)
print "  e: ", sn, " ", cn, " ", dn, "\n"
