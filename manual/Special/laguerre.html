Laguerre functions

The generalized Laguerre polynomials are defined in terms of confluent hypergeometric functions as L^a_n(x) = ((a+1)_n / n!) 1F1(-n,a+1,x).
The Laguerre polynomials are L_n(x) = L^0_n(x)

Module Laguerre

GSL::Special::Hyperg::L1
GSL::Special::Hyperg::L1_e
Hyperg::L1(a, x) -> d1
a and x are doubles. These routines evaluate the generalized Laguerre polynomials L^a_1(x), L^a_2(x), L^a_3(x) using explicit representations

GSL::Special::Hyperg::Ln
GSL::Special::Hyperg::Ln_e
Hyperg::Ln(n, a, x) -> d1
n is an integer, a and x are doubles. These routines evaluate the generalized Laguerre polynomials L^a_n(x) for a > -1, n >= 0.
