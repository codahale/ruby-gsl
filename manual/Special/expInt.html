Exponential Integrals

Information on the exponential integrals can be found in Abramowitz & Stegun, Chapter 5.

Exponential Integral

GSL::Special::Exp::E1
GSL::Special::Exp::E1_e
Exp::E1(d1) -> d2
These routines compute the exponential integral E_1(x), E_1(x) := Re \int_1^\infty dt \exp(-xt)/t.

GSL::Special::Exp::E2
GSL::Special::Exp::E2_e
Exp::E2(d1) -> d2
These routines compute the second-order exponential integral E_2(x), E_2(x) := \Re \int_1^\infty dt \exp(-xt)/t^2.

Ei(x)

GSL::Special::Exp::Ei
GSL::Special::Exp::Ei_e
Exp::Ei(d1) -> d2
These routines compute the exponential integral E_i(x), Ei(x) := - PV(\int_{-x}^\infty dt \exp(-t)/t) where PV denotes the principal value of the integral.

Hyperbolic Integrals

GSL::Special::Exp::Shi
GSL::Special::Exp::Shi_e
Exp::Shi(d1) -> d2
These routines compute the integral Shi(x) = \int_0^x dt \sinh(t)/t.

GSL::Special::Exp::Chi
GSL::Special::Exp::Chi_e
Exp::Chi(d1) -> d2
These routines compute the integral Chi(x) := Re[ \gamma_E + \log(x) + \int_0^x dt (\cosh[t]-1)/t] , where \gamma_E is the Euler constant (available as the constant Math::M_EULER).

Ei_3(x)

GSL::Special::Exp::Ei3
GSL::Special::Exp::Ei3_e
Exp::Ei3(d1) -> d2
These routines compute the exponential integral Ei_3(x) = \int_0^x dt \exp(-t^3) for x >= 0.

Trigonometric Integrals

GSL::Special::Exp::Si
GSL::Special::Exp::Si_e
Exp::Si(d1) -> d2
These routines compute the Sine integral Si(x) = \int_0^x dt \sin(t)/t.

GSL::Special::Exp::Ci
GSL::Special::Exp::Ci
Exp::Ci(d1) -> d2
These routines compute the Cosine integral Ci(x) = -\int_x^\infty dt \cos(t)/t for x > 0.

Arctangent Integral

GSL::Special::Exp::atanint
GSL::Special::Exp::atanint_e
Exp::atanint(d1) -> d2
These routines compute the Arctangent integral AtanInt(x) = \int_0^x dt \arctan(t)/t.
