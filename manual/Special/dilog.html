Real Argument

GSL::Special::Dilog::dilog
GSL::Special::Dilog::dilog_e
Dilog::dilog(d1) -> d2
These routines compute the dilogarithm for a real argument. In Lewin's notation this is Li_2(x), the real part of the dilogarithm of a real x. It is defined by the integral representation Li_2(x) = - \Re \int_0^x ds \log(1-s) / s. Note that \Im(Li_2(x)) = 0 for x <= 1, and -\pi\log(x) for x > 1.

Complex Argument

GSL::Special::Dilog::complex_dilog_e
complex_dilog_e(d1, d2) -> anArray
This function computes the full complex-valued dilogarithm for the complex argument z = r \exp(i \theta); it returns the bidimensional array anArray, whose components are Result objects representing the real an imaginary part.
