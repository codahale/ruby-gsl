Airy Functions and Derivatives

The Airy functions Ai(x) and Bi(x) are defined by the integral representations, 
Ai(x) = (1/\pi) \int_0^\infty \cos((1/3) t^3 + xt) dt
Bi(x) = (1/\pi) \int_0^\infty (e^(-(1/3) t^3) + \sin((1/3) t^3 + xt)) dt
They satisfy the differential equation
d^2 f/dx^2 - x f = 0
For further information see Abramowitz & Stegun, Section 10.4.

GSL::Special::Airy::Ai_scaled

Airy::Ai_scaled(d1, mode) -> d2
These routines compute a scaled version of the Airy function S_A(x) Ai(x). For x>0 the scaling factor S_A(x) is \exp(+(2/3) x^(3/2)), and is 1 for x<0.

GSL::Special::Airy::Bi_scaled
GSL::Special::Airy::Bi_scaled_e
These routines compute a scaled version of the Airy function S_B(x) Bi(x). For x>0 the scaling factor S_B(x) is exp(-(2/3) x^(3/2)), and is 1 for x<0.

require 'assert'
require "GSL"
include GSL::Special; include Math
require 'gsl_util'

printf "%.15f\n", Airy::Ai(0, PREC_DOUBLE) # 0.355028053887817
assert Airy::Ai(0, PREC_DOUBLE) == Airy::Bi(0, PREC_DOUBLE)/sqrt(3)

puts "precision modes"
x = 3.0
rd =  Airy::Ai(x, PREC_DOUBLE)
p rd # 0.006591139357
rs =  Airy::Ai(x, PREC_SINGLE); p rs - rd # 8.630422765e-14
ra =  Airy::Ai(x, PREC_APPROX); p ra - rd # 8.630422765e-14

r = Airy::Ai_e(x, PREC_DOUBLE)
printf "%g (err = %g)\n", r.val, r.err # 0.00659114 (err = 1.08527e-17)
assert r.class == GSL::Result

rbi =  Airy::Bi(x, PREC_DOUBLE)
p rbi # 14.03732896

r = Airy::Bi_e(x, PREC_DOUBLE)
printf "%g (err = %g)\n", r.val, r.err # 14.0373 (err = 2.29245e-14)

rs = Airy::Ai_scaled(x, PREC_DOUBLE)
assert rs =~ [rd*exp(2.0/3 * pow(x,3.0/2)), 1.0e-16]

rs = Airy::Bi_scaled(x, PREC_DOUBLE)
assert rs =~ [rbi*exp(-2.0/3 * pow(x,3.0/2)), 1.0e-16]
puts "derivatives of Airy functions"
x = 0.4
p Airy::Ai_deriv(x, PREC_DOUBLE)		  
p Airy::Bi_deriv(x, PREC_DOUBLE)
p Airy::Ai_deriv_scaled(x, PREC_DOUBLE)
p Airy::Bi_deriv_scaled(x, PREC_DOUBLE)

puts "Zeros of Airy Functions"
z1 = Airy::zero_Ai(1) 
assert Airy::Ai(z1, PREC_DOUBLE) =~ [0, 1.0e-16]
z2 = Airy::zero_Bi(1)
assert Airy::Bi(z2, PREC_DOUBLE) =~ [0, 1.0e-16]
z3 = Airy::zero_Ai_deriv(1)
assert Airy::Ai_deriv(z3, PREC_DOUBLE) =~ [0, 1.0e-16]
z4 = Airy::zero_Bi_deriv(1)
assert Airy::Bi_deriv(z4, PREC_DOUBLE) =~ [0, 1.0e-16]
