Module Synchrotron

S1 ,  S2  

GSL::Special::Synchrotron::S1
GSL::Special::Synchrotron::S1_e
Synchrotron::S1(x) -> d1
x is a double. These routines compute the first synchrotron function x \int_x^\infty dt K_{5/3}(t) for x >= 0.

GSL::Special::Synchrotron::S2
GSL::Special::Synchrotron::S2_e
Synchrotron::S2(x) -> d1
These routines compute the second synchrotron function x K_{2/3}(x) for x >= 0. 

require "GSL"
include GSL
include GSL::Special
include Math

puts "S1"
x = 42.0
r = Synchrotron::S1(x)
p r
r = Synchrotron::S1_e(x)
p r.val; p r.err

puts "S2"
x = 42.0
r = Synchrotron::S2(x)
p r
