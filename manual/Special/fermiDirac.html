Complete Fermi-Dirac Integrals

The complete Fermi-Dirac integral F_j(x) is given by,
F_j(x) := (1/r\Gamma(j+1)) \int_0^\infty (t^j / (\exp(t-x) + 1)) GSL::Special::FermiDirac::Fm1
GSL::Special::FermiDirac::Fm1_e
Exp::Fm1(d1) -> d2
These routines compute the complete Fermi-Dirac integral with an index of -1. This integral is given by F_{-1}(x) = e^x / (1 + e^x).

GSL::Special::FermiDirac::F0
GSL::Special::FermiDirac::F0_e
FermiDirac::F0(d1) -> d2
These routines compute the complete Fermi-Dirac integral with an index of 0. This integral is given by F_0(x) = \ln(1 + e^x).

GSL::Special::FermiDirac::F1
GSL::Special::FermiDirac::F1_e
FermiDirac::F1(d1) -> d2
These routines compute the complete Fermi-Dirac integral with an index of 1, F_1(x) = \int_0^\infty dt (t /(\exp(t-x)+1)).

GSL::Special::FermiDirac::F2
GSL::Special::FermiDirac::F2_e
FermiDirac::F2(d1) -> d2
These routines compute the complete Fermi-Dirac integral with an index of 2, F_2(x) = (1/2) \int_0^\infty dt (t^2 /(\exp(t-x)+1)).

GSL::Special::FermiDirac::Fint
GSL::Special::FermiDirac::Fint_e
FermiDirac::Fint(j, d1) -> d2
j is an integer. These routines compute the complete Fermi-Dirac integral with an integer index of j, F_j(x) = (1/\Gamma(j+1)) \int_0^\infty dt (t^j /(\exp(t-x)+1)).

Incomplete Fermi-Dirac Integrals

The incomplete Fermi-Dirac integral F_j(x,b) is given by,
F_j(x,b) := (1/\Gamma(j+1)) \int_b^\infty (t^j / (\Exp(t-x) + 1))

GSL::Special::FermiDirac::Finc_0
GSL::Special::FermiDirac::Finc_0_e
FermiDirac::Finc_0(d1, d2) -> d3
These routines compute the incomplete Fermi-Dirac integral with an index of zero, F_0(x,b) = \ln(1 + e^{b-x}) - (b-x).
