Riemann Zeta Function

The Riemann zeta function is defined by the infinite sum \zeta(s) = \sum_{k=1}^\infty k^{-s}.

GSL::Special::Zeta::zeta_int
GSL::Special::Zeta::zeta_int_e
Zeta::zeta_int(n) -> d1
These routines compute the Riemann zeta function \zeta(n) for integer n, n \ne 1.

GSL::Special::Zeta::zeta
Zeta::zeta(s) -> d1
s is a double. These routines compute the Riemann zeta function \zeta(s) for arbitrary s, s \ne 1.

Hurwitz Zeta Function

The Hurwitz zeta function is defined by \zeta(s,q) = \sum_0^\infty (k+q)^{-s}.

GSL::Special::Zeta::hzeta
Zeta::hzeta(s, q) -> d1
s and q are doubles. These routines compute the Hurwitz zeta function \zeta(s,q) for s > 1, q > 0.

Hurwitz Zeta Function

The Hurwitz zeta function is defined by \zeta(s,q) = \sum_0^\infty (k+q)^{-s}. These routines compute the Hurwitz zeta function \zeta(s,q) for s > 1, q > 0.

Eta Function

The eta function is defined by \eta(s) = (1-2^{1-s}) \zeta(s).
